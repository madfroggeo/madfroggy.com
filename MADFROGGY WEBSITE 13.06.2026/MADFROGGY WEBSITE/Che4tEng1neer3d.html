CHE4T ENG1NEER3D

HACK IT. OVERPOWER IT.

winner of the GDevelop BIG game jam #7, CHE4T ENG1NEER3D is a a cool top-down, 8-bit game where you hack the game to gain the upper hand. This game was made for the GDevelop BIG Game Jam #7. I think that overall, I did pretty good for my first game jam as I was able to produce a finished product that I'm very happy with. Make sure to rate the game, and I hope that all of you enjoy!

Also thank you to FunnyBringus for making all of the music for the game!
